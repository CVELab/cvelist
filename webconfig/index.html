Go Vulnerability Database

This is the Go vulnerability database. It is meant to be accessed by programs that want to find vulnerabilities associated with Go modules.

The database's API is documented at go.dev/security/vuln/database. To simply browse the reports in the database, use pkg.go.dev/vuln.

See go.dev/security/vuln to learn about Go's Vulnerability Management system, including details on how the database is populated, and client libraries and tools that can be used to fetch information from the database.

To learn about vulncheck, the package which allows developers to scan their projects' dependencies to detect public vulnerabilities, see go.dev/security/vuln/vulncheck.

See vuln.go.dev/privacy for privacy information about this service.

See vuln.go.dev/copyright for copyright and licensing information.
